What is FZMA BOT?

The FZMA BOT is an API for custom commands on most chat bots. Commands can be setup freely to anything, or default command as !fzma or !h1` to easily and quickly enter the extra command/trigger or text to search. Enter an items name

Availability

With chat bot integration on the rise FZMA BOT is available for various chat bot therefor chats. Here are known and tested Chat Bots:

  • Nightbot
  • Moobot
  • Revlo
  • Vivbot

Known and tested chats:

  • Discord
  • Twitch
  • YouTube
  • ReStream.io

How to use?

Some examples of basic commands are:

!fzma help
!fzma ar15
!fzma medkit

Manual Setup

To setup manually on Nightbot, Moobot, etc by adding a custom command on the website. Click Add Command, then in the Command field type !fzma or !kotk or anything you'd like. Then in the Message field enter $(customapi http://h1z1fzma.com/api/kotk.php?bot_input=$(querystring))

Direct Setup

For nightbot/moobot/etc enter 

!addcom !fzma $(customapi http://h1z1fzma.com/api/kotk.php?bot_input=$(querystring))
